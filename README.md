# 🏠 House Price Prediction System

A Machine Learning web application that predicts house prices based on various house features such as area, bedrooms, bathrooms, parking, furnishing status, and more.

Built using:
- Python
- Scikit-learn
- Streamlit
- Pandas
- NumPy

---

# 🚀 Live Demo

https://ml-house-price-sys.streamlit.app

---

# 📌 Features

✅ Predict house prices instantly  
✅ Uses Machine Learning regression model  
✅ Professional Streamlit web interface  
✅ Supports multiple house features  
✅ Interactive UI with dropdowns and animations  
✅ Real-time predictions

# 📂 Dataset Features

The model uses the following features:

- Area
- Bedrooms
- Bathrooms
- Stories
- Main Road Access
- Guest Room
- Basement
- Hot Water Heating
- Air Conditioning
- Parking
- Preferred Area
- Furnishing Status

# 📊 Model Used

Linear Regression

Reason for using Linear Regression:
- Simple and beginner-friendly model
- Fast training and prediction
- Useful for understanding regression concepts
- Good baseline model for house price prediction
